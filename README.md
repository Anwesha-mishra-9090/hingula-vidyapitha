# Hingula Vidya Pitha - School Website

Government aided high school under Board of Secondary Education, Odisha. Established in 1992.

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Routing**: TanStack Router
- **Styling**: Tailwind CSS + shadcn/ui
- **Animations**: Framer Motion
- **Build Tool**: Vite
- **Package Manager**: Bun

## Features

- 📱 Fully responsive design
- 🌓 Dark/Light theme
- ⌨️ Command palette (⌘K) for quick navigation
- 📄 PDF generation for notices
- 🖼️ Gallery with lightbox
- 📊 Interactive charts
- 🔍 Search functionality
- 📝 Alumni directory with filters
- 🎓 Faculty directory with photos
- 📅 Academic calendar
- 📢 Live notice ticker

## Development

### Prerequisites

- Bun installed on your system

### Installation

```bash
# Install dependencies
bun install

# Start development server
bun dev

# Build for production
bun run build

# Preview production build
bun preview
```
